# Israeli Domain Blocker

## Description

This extension blocks websites associated with Israeli entities to protest against the occupation regime. 

## Features

- **Pre-loaded Domains:** 85 domains are already loaded when the extension is installed.
- Block Israeli domains easily.
- Whitelist functionality to allow access to certain domains.
- User-friendly interface resembling the Palestinian flag colors.

## How to Install

1. Download the ZIP file from this repository
2. Extract the ZIP file to a folder on your computer
3. Open Chrome/Edge/Brave and go to the Extensions page (`chrome://extensions/` or `edge://extensions/` or `brave://extensions/`)
4. Enable "Developer mode" in the top right corner
5. Click "Load unpacked" and select the extracted folder
6. The extension is now installed and ready to use

## How to Use

1. Enter a domain in the "Block Israeli Domains" input field and click "Block" to add it to the blocked list.
2. Use the "Whitelist" section to add domains you want to allow.
3. Remove domains from either list by clicking the respective "Remove" button.
